## Project Overview

This project is a React + Vite application for creating and reading branching stories in a choose-your-own-adventure format. The app includes story browsing, reading pages with choices, and authoring support.

## Local Development

```bash
npm install
npm run dev
```

Open the app at `http://localhost:5173` after starting the development server.

## Production Build

```bash
npm run build
npm run serve
```
